# 💳 Credit Card Financial Dashboard

## 📌 Overview
This project is a Power BI dashboard that analyzes credit card transactions and financial data to generate useful business insights.

---

## 🎯 Objective
- Understand customer spending behavior  
- Track revenue trends  
- Analyze transaction patterns  

---

## 🛠️ Tools Used
- Power BI  
- Excel  

---

## 📊 Features
- Revenue Analysis  
- Monthly Trends  
- Customer Insights  
- Interactive Visualizations  

---

## 📂 Files Included
- Dashboard file (.pbix)  
- Dataset (Excel/CSV)  

---

## 🚀 How to Use
1. Open the `.pbix` file in Power BI  
2. Explore the dashboard
